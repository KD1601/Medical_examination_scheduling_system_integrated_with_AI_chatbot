Yêu cầu:
	+ Máy tính đã cài đặt XAMPP	
	+ NodeJS (ver 14.17.0)
	+ IDE (Visual Studio Code hoặc các phần mềm khác)

Cách cài đặt và chạy hệ thống:
	B1: Mở XAMPP và bật 2 module Apache, MySQL
	B2: Mở phpMyAdmin trên MySQL
	B3: Import file booking_health.sql vào database của hệ thống phpMyAdmin
	B4: Mở lần lượt 2 thư mục Back-end và Front-end lên bằng VS Code
	B5: Bật terminal của cả 2 thư mục được mở và chạy câu lệnh 'npm install'
	B6: Sau khi đã cài đặt các thư viện thành công, lần lượt chạy câu lệnh 'npm start' cho cả 2 thư mục.
